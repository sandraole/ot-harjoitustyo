# Book Tracker

Book Tracker mahdollistaa pitämään kirjaa mitä kirjoja on lukenut ja seuraamaan omia lukutavoitteita. Käyttäjä voi lisätä kirjoja lukulistalle, merkitä kirjoja luetuksi, sekä seurata kuinka monta kirjaa ja sivua hän on lukenut. Sovelluksen tarkoitus on motivoida lukemaan enemmän, sekä muistamaan jo luetut kirjat.
